# Project Map

This directory contains selected graphify outputs for agent orientation.

- `GRAPH_REPORT.md` - graph audit report and suggested questions.
- `GRAPH_TREE.html` - file-oriented tree view.
- `graph.html` - interactive graph view.
- `graph.json` - raw graph data for graph-aware tools.

The map was generated from this repository with static code extraction plus
semantic extraction over docs, images, and metadata. Treat it as an orientation
aid, not as a replacement for checking the current source.
